# WaveDefiant 游戏设计文档 (GDD)

## 1. 游戏概述
**类型**：2D Roguelike 波次生存游戏
**视角**：俯视角 (Top-down)
**目标平台**：Steam PC / Nintendo Switch
**定价参考**：$4.99 - $9.99

## 2. 核心玩法
玩家在一个大型竞技场中移动，自动攻击涌来的敌人。击杀获得经验升级，每次升级随机三选一获得强化。波次递增，难度持续上升，每 5 波出现 Boss。

### 2.1 操作
- **移动**：WASD / 方向键 / 左摇杆
- **暂停**：ESC / Start 键

### 2.2 核心循环
```
移动 → 自动攻击 → 杀敌获XP → 升级选Buff → 更强 → 继续
```
